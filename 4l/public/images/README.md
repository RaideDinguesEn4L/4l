# 📸 PLACEZ VOS IMAGES ICI

Ce dossier doit contenir toutes vos images locales pour le site.

## Structure requise :

```
images/
├── hero/
│   └── desert-hero.jpg
├── team/
│   ├── thomas.jpg
│   └── marie.jpg
├── car/
│   ├── 4l-main.jpg
│   └── desert-background.jpg
└── blog/
    ├── acquisition.jpg
    ├── mecanique.jpg
    ├── partenaires.jpg
    └── depart.jpg
```

## 📏 Dimensions recommandées :

- **desert-hero.jpg** : 1920x1080px (paysage désert)
- **thomas.jpg, marie.jpg** : 400x400px (carré)
- **4l-main.jpg** : 800x600px (votre 4L)
- **desert-background.jpg** : 1920x1080px (fond désert)
- **blog/*.jpg** : 600x400px (articles)

## 🔧 Comment ajouter vos images :

1. Créez les sous-dossiers : `hero`, `team`, `car`, `blog`
2. Ajoutez vos images avec les noms exacts indiqués
3. Format JPG recommandé (compression 80-85%)
4. Optimisez vos images avant de les ajouter

## ⚠️ Placeholders temporaires

En attendant vos vraies images, vous pouvez :
- Utiliser des images de test avec les mêmes noms
- Le site fonctionnera une fois les images en place
- Next.js optimisera automatiquement les images
